# VRC World Opener

X（Twitter）で「#VRChat_World紹介」タグの付いたポストで、URLが書かれていなくてそのワールドに行くのにコピペして検索！ということ、よくありますよね？
そんな時に、ワンクリックでワールドページに飛べるようにするChrome拡張機能です。

## ⚠️ 注意事項（使用前に必ずお読みください）

前提：本ツールはchrome ウェブストアでまだ公開していない**試作品**です。予期しない不具合が発生する可能性がありますので、**自己責任**の上ご使用ください。

- **X（Twitter）の仕様変更について**
  - Xの仕様変更により、正常に動作しなくなる可能性があります
  - 表示が崩れたり不具合が生じた場合は、本拡張機能を無効にするか削除してください
- **VRChat APIについて**
  - VRChat APIの仕様変更により、**正常に動作しなくなる可能性**があります
- **ワールド名の検出について**
  - **ワールド名の表記方法は人により様々**なため、ワールド名が正しく検出できずボタンが表示されない場合があります。
  - また、現状は**ワールド作者名を検索条件に含めていない**ため、**稀に同名の別作者ワールドページが表示される**ことがあります。
    その場合は「ワールド名で検索」ボタンから正しいワールドを選択してください。
  - 検出できそうなのにできていないパターンや、明らかにおかしい挙動があれば、該当のポストを報告いただければ可能な範囲で対応します
  （ワールド名の区切りが特定できない場合など、機械的に検出できないものは対応が難しいです）
 ただし、検出は割と力技なので**完璧ではないことをご理解ください**。
- **LLM利用について**
  - 本拡張機能の作成にはLLMを使用しています。致命的な問題があればご報告ください（VibeCoding習作）

## 機能

### 🔍 ワールド名を自動検出

ポスト内のワールド名を自動で見つけて、ボタンを表示します。
現状は以下のような書き方に対応しています。
追加して欲しい物があれば対応します。

| パターン     | 例                                                                        |
| ------------ | ------------------------------------------------------------------------- |
| コロン形式   | `World: ワールド名`、`ワールド: ◯◯`、`ワールド名: ◯◯`           |
| 括弧形式     | `World『ワールド名』`、`World「ワールド名」`、`World【ワールド名】` |
| 絵文字形式   | `🌐ワールド名`、`🌍ワールド名`、`🌎ワールド名`、`🌏ワールド名`    |
| 英語形式     | `World name: ◯◯`                                                      |
| 作者付き形式 | `ワールド名`（改行）`By 作者名`                                       |

### 🔗 ワールドを開くボタン

クリックするだけで、VRChatのワールドページを直接開けます。
（初回のみVRChatへのログインが必要です）

### 🔎 ワールド名で検索ボタン

ワンクリックでVRChatの検索ページを検索された状態で開きます。
（ログイン不要です）

### ⚙️ お好みで設定可能

上記のボタンの表示/非表示を個別に切り替えられます。

## インストール方法

1. このリポジトリをダウンロードまたはクローン
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をオン
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `vrchat-world-opener` フォルダを選択

## 使い方

### 基本的な使い方

1. X（Twitter）で `#VRChat_World紹介` を含むポストを閲覧
2. ワールド名が検出されると、ポストにボタンが表示される
3. 「🔗 ワールドを開く」または「🔍 ワールド名で検索」をクリック

### VRChatへのログイン（ワールドを開くボタンを使う場合）

1. 拡張機能アイコンをクリックしてポップアップを開く
2. VRChatのユーザー名とパスワードでログイン
3. 2FAを設定している場合は認証コードを入力

### 詳細設定

ポップアップ内の詳細設定から以下をカスタマイズできます：

- 拡張機能の有効/無効
- 「ワールドを開く」ボタンの表示/非表示
- 「ワールド名で検索」ボタンの表示/非表示

## セキュリティについて

### 🔒 パスワードは保存されません

この拡張機能は **パスワードを一切保存しません**。

- ログイン時にパスワードはVRChat APIに直接送信され、**即座にメモリから破棄**されます
- ローカルストレージに保存されるのは「ログイン状態」と「表示名」のみです
- ログイン後はVRChatが発行するセッションCookieで認証を行います

### 🔐 通信は暗号化されています

- すべてのAPI通信は **HTTPS（TLS暗号化）** で行われます
- 通信先は `https://api.vrchat.cloud` と `https://vrchat.com` のみで、それ以外のサイトには一切通信しません

### 🛡️ 使用している権限

本拡張機能が使用するブラウザ権限は以下のみです：

| 権限          | 用途                                        |
| ------------- | ------------------------------------------- |
| `storage`   | 設定（有効/無効、ボタン表示設定など）の保存 |
| `activeTab` | 現在開いているタブへのアクセス              |
| `tabs`      | X（Twitter）のタブに設定変更を通知          |

- X（Twitter）以外のサイトでは動作しません
- 閲覧履歴やブックマークなどへのアクセス権限は**一切ありません**
